== Installation ==



直接在后台搜索 mydownload 在线安装启用即可，无需任何设置和代码修改！





== Frequently Asked Questions ==



常见问题：



1.如果遇到与其他插件位置冲突,请改变插件激活次序

2.如果需要自定义插件模板文件，请到插件目录下的TP文件夹中进行修改

3.如果因为页面框架大小影响文章页面下载按钮的样式 可以在后台选择按钮部分的大小来解决问题！
